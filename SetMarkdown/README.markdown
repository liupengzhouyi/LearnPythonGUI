
# Set MarkDown
> 主要作用：
> * 一个文件夹中有多个markdown文件
> * 把多个markdown文件合成一个文件


## 主要技术

* Python GUI编程(Tkinter)

* 文件读取

## 效果展示

## 原理

1. give me a path
2. 读取路径下所有文件
3. 按照文件名排序
4. create a new file to get all data
5. rading every files data 
6. write data to new file
